# 后台管理联调验收清单（blog_project + blog_vue）

## 0. 启动与环境确认

- [ ] 启动后端：`cd /home/pdnbplus/project/blog-project/blog_project && .venv/bin/python manage.py runserver 0.0.0.0:8001`
- [ ] 启动前端：`cd /home/pdnbplus/project/blog_vue/blog_vue && npm run dev`（默认 `5174`）
- [ ] 访问前端地址：`http://127.0.0.1:5174`
- [ ] 确认前端请求后端地址为：`http://127.0.0.1:8001/api/v1`

## 1. 登录与权限

- [ ] 管理员账号可登录并进入后台管理页面
- [ ] 非管理员访问管理接口时被拒绝（401/403）

## 2. 文章管理核心流程

- [ ] 列表首次加载正常（无白屏、无报错）
- [ ] 分页切换与每页条数切换正常
- [ ] 关键词查询（标题/摘要/slug）结果正确
- [ ] 状态筛选（draft/published/archived）结果正确
- [ ] 新建文章成功后列表可见
- [ ] 编辑文章成功后列表内容更新
- [ ] 归档文章成功后状态变更为 `archived`
- [ ] 新建或编辑时使用重复 `slug`，返回提示：`slug 已存在，请更换`

## 3. 前台联动与缓存相关验证

- [ ] 执行后台新建/编辑/归档后，前台首页与文章列表刷新可看到最新数据
- [ ] 分类树接口数据正确，分类展示与后台一致
- [ ] 无 Redis 依赖下，接口缓存行为正常（管理端写操作后公共接口缓存失效）

## 4. 异常与稳定性

- [ ] 人为制造后端错误时，前端可展示错误提示且页面不崩溃
- [ ] 连续执行“新建 -> 编辑 -> 归档 -> 查询”至少一轮，状态流转正确
